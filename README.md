# 每日计划打卡应用

一个基于PyQt5的个人打卡管理工具，支持任务管理、打卡记录和完成情况记录。

## 功能特点

- **任务管理**：添加、修改、删除每日任务
- **打卡记录**：记录每天的任务完成情况
- **完成情况**：为每个任务记录详细的完成说明
- **日历视图**：可视化展示打卡历史，完成的日期会高亮显示
- **护眼界面**：采用护眼配色方案，保护视力
- **字体调节**：支持8-36px字体大小调节，适应不同需求
- **数据持久化**：所有数据保存在本地JSON文件中

## 界面截图

### 主界面
- 设置区域：初始日期和字体大小设置
- 今日任务：显示当天需要完成的任务列表
- 完成情况：每个任务后面有编辑框可记录完成情况
- 打卡日历：显示打卡历史，绿色高亮表示已完成

## 安装要求

- Python 3.6+
- PyQt5

## 安装依赖

```bash
pip install PyQt5
```

## 使用方法

### 运行应用

1. 直接运行Python脚本：
   ```bash
   python main.py
   ```
   
   或使用批处理文件：
   ```bash
   start_app.bat
   ```

### 打包成EXE文件

1. 运行打包脚本：
   ```bash
   build.bat
   ```

   该脚本会自动：
   - 检查并安装PyInstaller（如果未安装）
   - 清理旧的打包文件
   - 使用PyInstaller打包成单个exe文件
   - 打包完成后自动打开dist文件夹

2. 打包完成后，可执行文件位于：
   ```
   dist\每日计划打卡.exe
   ```

3. 分发：
   - 只需分发exe文件，无需安装Python环境
   - 用户直接双击exe文件即可运行

2. 设置初始日期：在顶部设置区域输入开始打卡的日期

3. 添加任务：点击"添加任务"按钮，输入任务名称

4. 打卡：勾选任务完成，在编辑框中记录完成情况

5. 查看历史：在日历中点击日期，查看当天的完成情况

## 数据文件

- `clock_in_data.json`：存储所有任务和打卡数据

## 技术栈

## 界面特点

- 护眼配色：浅蓝色背景，绿色任务区域，黄色日历区域
- 响应式字体：所有内容随字体设置动态调整
- 紧凑布局：优化空间利用，减少无效空白
- 美观按钮：大尺寸按钮，清晰易操作
